# Key Features

- Home Page
- Video Listing Page
- Playlist Management
- Like/Dislike
- Watch Later
- History
- Authentication
  - Sign-up
  - Login
  - Logout
- Single Video Page
- Notes on a video
- Filter Videos with Category
- Loading & Alerts
